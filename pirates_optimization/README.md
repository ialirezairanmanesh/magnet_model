# بهینه‌سازی مدل MAGNET با الگوریتم Pirates

این پروژه از الگوریتم بهینه‌سازی Pirates برای بهینه‌سازی پارامترهای مدل MAGNET استفاده می‌کند.

## ساختار فایل‌ها

- `magnet_objective.py`: تابع هدف برای بهینه‌سازی پارامترهای مدل
- `optimize_magnet.py`: فایل اصلی اجرای الگوریتم
- `pirates.py`: پیاده‌سازی الگوریتم Pirates
- `functions.py`: توابع کمکی
- `selection.py`: روش‌های انتخاب
- `helpers.py`: توابع کمکی برای رسم نمودار
- `bcolors.py`: رنگ‌های ترمینال

## پارامترهای بهینه‌سازی

الگوریتم پارامترهای زیر را بهینه می‌کند:

1. `embedding_dim`: بعد embedding (32 تا 256)
2. `num_heads`: تعداد headهای attention (2 تا 16)
3. `num_layers`: تعداد لایه‌های transformer (2 تا 8)
4. `dim_feedforward`: بعد لایه feedforward (128 تا 1024)
5. `dropout`: نرخ dropout (0.1 تا 0.5)

## نحوه اجرا

برای اجرای الگوریتم، کافیست فایل `optimize_magnet.py` را اجرا کنید:

```bash
python optimize_magnet.py
```

## تنظیم پارامترهای الگوریتم

می‌توانید پارامترهای الگوریتم را در فایل `optimize_magnet.py` تغییر دهید:

- `num_ships`: تعداد کشتی‌ها (پیش‌فرض: 20)
- `max_iter`: تعداد تکرارها (پیش‌فرض: 50)
- `top_ships`: تعداد کشتی‌های برتر (پیش‌فرض: 5)
- `max_wind`: حداکثر سرعت باد (پیش‌فرض: 1.0)
- `max_r`: حداکثر شعاع دید (پیش‌فرض: 1.0)
- `p_hr`: احتمال طوفان (پیش‌فرض: 0.1)

## خروجی

الگوریتم پس از اجرا، بهترین پارامترهای پیدا شده و خطای مربوطه را نمایش می‌دهد.

## نکات مهم

1. الگوریتم ممکن است زمان زیادی برای اجرا نیاز داشته باشد.
2. می‌توانید تعداد تکرارها را بر اساس نیاز خود تغییر دهید.
3. در صورت نیاز به تغییر محدوده پارامترها، فایل `optimize_magnet.py` را ویرایش کنید.
